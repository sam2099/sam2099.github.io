shell （四） 字符串

发表于 2015-12-28 | 分类于 linux |

主要内容

  • 字符串拼接
  • 字符串长度
  • 字符串截取
  • 字符串查找

字符串是shell编程中最常用最有用的数据类型，字符串可以用单引号，也可以用双引号，也可以不用引号。

单引号和双引号的区别 :

  • 单引号之间的内容成为纯文本，取消所有特殊符号的意义；
  • 双引号仅特殊符号 $ \ ` 这三个的含义保留。
