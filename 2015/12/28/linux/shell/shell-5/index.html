shell （五） 数组

发表于 2015-12-28 更新于 2022-12-08 分类于 linux

主要内容
bash支持一维数组（不支持多维数组），并且没有限定数组的大小。数组元素的下标由0开始编号。获取数组中的元素要利用下标，下标可以是整数或算术表达式，其值应大于或等于0。
